# Todo-MVC
类 FLux 框架快速上手项目, 使用 React + Flux Utils / redux / react-redux / mobx 实现多个简单应用

### Counter
- Counter 的行为很简单, 点击 + 或 - 按钮进行计数

### Todos
- Todos 有以为行为

    - 点击输入框进行输入, 回车或者点击输入框外部完成输入, 创建新条目
    - 点击条目前的 ✅ 切换条目的完成状态
    - 双击条目进行编辑
    - 点击条目后的 ❎ 删除条目
    - 底部实时更新未完成的条目
    - 存在已完成的条目时, 底部显示 清除全部已完成条目 按钮
